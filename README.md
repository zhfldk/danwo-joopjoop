
# 이미지 → 단어장 생성기 (Rounded UI, PDF 저장)

클라이언트(브라우저)만으로 동작하는 단어장 생성기입니다.

## 주요 기능
- 이미지 여러 장 업로드 → **Tesseract.js**로 OCR
- **단어-뜻** 형태가 이미지에 있으면 그대로 사용
- 뜻이 없거나 비어있으면 → **dictionaryapi.dev**로 자동 정의 채움
- 표에서 **수정/삭제**
- **중복 정리**, **CSV 내보내기**
- **프린트(브라우저 인쇄)** 및 **PDF 저장** (리스트/플래시카드/워크시트)

## 사용 방법
1. `index.html`을 열어 사용합니다. (권장: 로컬 서버 또는 Vercel/Netlify/GitHub Pages에 배포)
2. 이미지 업로드 → **OCR 시작**
3. 결과 표 확인 후, 비어있는 뜻은 **뜻 채우기(사전)** 버튼
4. **CSV 내보내기** 또는 **PDF 저장** / **인쇄**

## 배포
- 정적 사이트이므로 GitHub Pages, Netlify, Vercel에서 **무료 배포** 가능

## 주의사항
- 사전/번역 API는 공개 무료 서비스(dictionaryapi.dev)를 사용합니다.
- 한국어 자동 번역은 기본 비활성화(간단 태그)이며, 상용 번역 API(Papago/Google/Kakao)를 붙일 수 있습니다.
